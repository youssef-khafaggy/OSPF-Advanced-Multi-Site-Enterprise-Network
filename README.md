# OSPF Advanced: Multi-Site Enterprise Network

## Overview
An advanced, production-grade enterprise network topology designed and implemented to showcase scalable multi-site routing architectures, secure neighbor adjacencies, and intelligent first-hop gateway redundancy frameworks.

## Features
- **Multi-Area OSPF Architecture** (Area 0, Area 1, Area 2)
- **OSPF MD5 Authentication** (Secure Adjacency)
- **OSPF Virtual-Link** (Connecting Discontinuous Area 2)
- **EIGRP (AS 100) & RIP Domains**
- **Multi-Protocol Route Redistribution**
- **Precise Route-Map Filtering** (Denying unauthorized subnets)
- **VRRP Gateway Redundancy** (Virtual Router Redundancy Protocol)
- **IP SLA & Object Tracking** (Intelligent link monitoring)
- **Dynamic Decrement Priority Failover**
- **Centralized DHCP Routing**
- **Stable Loopback Management IDs**
- **VMware Integration** (Real Host VMnet8 Bridging)
